## SOP File Routing (Drop into any project instructions)

Before writing ANY file to the SOP Hub, confirm the correct paths. Do NOT guess or infer from project instructions alone -- the canonical structure is below.

### Naming Convention
- Prescott Epoxy: `PEC-[DEPT]-[###] [Title].docx`
- Finishing Touch: `FTP-[DEPT]-[###] [Title].docx`
- Department codes: SALES, OPS, FIN, ADMIN, SAFETY

### Before Filing a New SOP
1. List existing files in the target folder to find the next available number
2. Use "PEC" not "PE" for Prescott Epoxy
3. Use "SOP Hub" (with space) not "SOP-Hub" (with hyphen)
4. Never reuse a retired number
5. Save .docx only (never .md) for SOPs going to Google Drive

### Common Mistakes to Avoid
- Writing to `SOP-Hub/` (hyphen) instead of `SOP Hub/` (space)
- Using `PE/` instead of `PEC/` for the company folder
- Starting numbering at 001 without checking what already exists
- Creating new subfolder structures that don't match the above
